# 链模式（责任链模式）

  1. 定义：实现链式调用，通过在自身方法中返回自身的方式，在一个对象连续多次调用自身方法

  2. 例子

var obj={
    a:fucntion(){
        console.log('aaa')
        // this返回自身obj
        return this
    },
    b:fucntion(){
        console.log('bbb')
        return this
    },
}

// 没有 rerun this
// obj.a().b()//报错
// 相当于
// var a=obj.a()
// obj.a()
